VMware에 Windows 10 설치 및 설정

VMware Workstation에서 Windows 10 가상 머신을 생성하고 설치한 뒤, 실습 환경에 맞게 기본 설정을 진행하는 과정을 정리하였다.

1. 실습 환경

- 가상화 프로그램: VMware Workstation Pro 25H2
- 설치 OS: Windows 10 x64
- ISO 파일: ko_windows_10_business_edition...
- 가상머신 이름: w10
- CPU: 1 Processor / 2 Core
- Memory: 2GB
- Disk: 50GB
- Network: NAT
- IP 주소: 10.0.0.101
- Subnet Mask: 255.255.255.0
- Gateway: 10.0.0.254
- DNS: 168.126.63.1 / 8.8.8.8


2. Windows 10 가상 머신 생성

  • 새 가상 머신 생성

    VMware Workstation을 실행한 후 File - New Virtual Machine...을 선택하여 새 가상머신을 생성한다.


  • Custom 옵션 선택

    가상 머신 설정을 직접 지정하기 위해 Custom(advanced)를 선택한다.
    Typical을 선택하면 기본값이 설정되지만, 실습에 맞게 CPU, 메모리 등을 설정하기 위해 Custom(advanced)를 선택하였다.


  • 하드웨어 호환성 설정

    Hardware compatibility는 기본값인 Workstation 25H2 or later로 설정하였다.


  • OS 설치 방식 선택

    ISO 파일은 나중에 직접 연결하기 위해 I will install the operating system later를 선택하였다.
    이 옵션을 선택하면 먼저 빈 가상 머신을 생성한 뒤, 설정 화면에서 ISO 파일을 연결할 수 있다.


  • Guest OS 선택

    Guest operating system은 Microsoft Windows를 선택하고, Version은 Windows 10 x64로 선택한다.


  • 가상 머신 이름 및 저장 위치 설정

    가상 머신 이름은 w10으로 지정하였다.
    저장 위치는 실습용 VM을 관리하기 위해 별도의 폴더인 D:\01_VMDK\w10로 설정하였다.


  • 펌웨어 타입 선택

    Firmware type은 BIOS를 선택하였다.


3. 가상 머신 하드웨어 설정

  • CPU 설정

    Processor는 1개, Core는 2개로 설정하였다.


  • 메모리 설정

    Memory는 2048MB(2GB)로 설정하였다.
    Windows 10 기본 설치 및 간단한 실습을 진행하기 위한 최소한의 메모리로 2GB를 할당하였다.


  • 네트워크 설정

    Network Type은 NAT로 선택하였다.


  • 디스크 컨트롤러 및 디스크 타입 선택

    I/O Controller Type은 기본 권장값인 LSI Logic을 선택하였고 Disk Type도 기본 권장값인 NVMe를 선택하였다.


  • 새 가상 디스크 생성

    새로운 Rocky Linux 설치를 위해 Create a new virtual disk를 선택하였다.


  • 디스크 용량 설정

    디스크 용량은 50GB로 설정하였고 Store virtual disk as a single file을 선택해 가상 디스크 파일을 하나의 파일로 저장하도록 설정하였다.


  • 불필요한 장치 제어

    가상 머신 생성 전 Customize Hardware를 클릭하여 하드웨어 설정을 확인해 실습에 필요하지 않은 Sound Card는 제거했다.


4. Windows 10 ISO 파일 연결



가상 머신 생성이 완료되면 Edit virtual machine settings를 클릭해 CD/DVD(SATA) - Use ISO image file을 선택하고 Windows 10 ISO 파일을 연결한다. 이후 Power on this virtual machine을 클릭하여 가상 머신을 실행한다.


5. Windows 10 설치

  • 설치 언어 선택 및 설치 시작

    Windows 설치 화면이 나타나면 설치할 언어, 시간 및 통화 형식, 키보드 입력 방식을 확인한다.
    기본값을 한국어 환경으로 두고 다음을 클릭한 후 지금 설치를 클릭하여 Windows 설치를 시작한다.


  • 운영체제 선택

    설치할 운영 체제는 Windows 10 Enterprise를 선택하고 다음을 클릭한다.


  • 라이선스 동의

    관련 통지 및 사용 조건 화면에서 동의함에 체크하고 다음을 클릭한다.


  • 설치 유형 선택

    새로운 가상 디스크에 Windows를 설치할 것이므로 사용자 지정: Windows만 설치(고급)을 선택한다.


  • 설치 위치 선택

    Windows를 설치할 위치로 드라이브 0 할당되지 않은 공간을 선택하고 다음을 클릭하면 Windows 설치가 진행된다.
    디스크 크기는 앞에서 설정한 50GB로 표시된다.


6. Windows 초기 설정

  • 지역 선택

    Windows 설치가 완료되면 초기 설정 화면이 나타난다.
    먼저 지역은 한국을 선택하고 키보드 백열은 Microsoft 입력기를 선택, 두번째 키보드 레이아웃은 필요하지 않으므로 건너뛰기를 클릭한다.


  • 네트워크 연결 화면

    네트워크 연결 단계에서 인터넷이 없는 상태로 설치를 진행하기 위해 왼쪽 아래의 인터넷이 없음을 클릭하고 Microsoft 계정도 연결 없이 로컬 계정으로 설치를 진행하기 위해 제한된 설치로 계속을 클릭한다.


  • 사용자 계정 생성 및 비밀번호 설정

    이 PC에서 사용할 사용자 이름을 Test로 입력하였다.
    원활한 실습을 위해 사용자 계정의 비밀번호를 입력하지 않고 다음을 클릭한다.


  • 개인 정보 및 서비스 설정

    Windows 초기 설정 중 위치 정보, 내 장치 찾기, 진단 데이터, 맞춤형 환경, 수동 입력 및 타이핑 개선, 광고 ID 사용 여부를 설정하는 화면이 나타난다.
    실습용 가상 머신에서는 이러한 개인화 및 부가 서비스 기능이 필요하지 않기 때문에 대부분 아니요를 선택하고, 진단 데이터는 최소 항목만 선택하여 진행하였다.


7. 컴퓨터 이름 변경

  • 시스템 속성 실행

    Windows 설치 후 컴퓨터 이름을 변경하기 위해 Windows+R을 눌러 실행 창을 열어 sysdm.cpl을 입력하고 확인을 클릭한다.


  • 컴퓨터 이름 변경 및 재시작 안내


    시스템 속성 창에서 컴퓨터 이름 탭을 확인한다.
    변경을 클릭한 뒤 컴퓨터 이름을 W10으로 변경하고 확인을 클릭한다.
    컴퓨터 이름 변경 후 재시작 안내 창이 나타나지만 추가 설정을 위해 바로 재시작을 하지 않고 나중에 다시 시작을 선택해주었다.


8. VMware Tools 설치

  • VMware Tools 설치 메뉴 실행

    VMware 상단 메뉴에서 VM - Install VMware Tools…를 클릭한다.
    VMware Tools를 설치하면 화면 해상도 자동 조정, 마우스 이동, 드라이버 인식 등이 더 원활해진다.


  • VMware Tools 드라이브 확인

    파일 탐색기에서 내 PC를 열면 DVD 드라이브(D:) VMware Tools가 연결된 것을 확인할 수 있다.
    해당 드라이브를 실행하여 VMware Tools 설치를 진행한다.


  • VMware Tools 설치 진행


    VMware Tools Setup 창이 나타나면 Next를 클릭, Setup Type은 기본값인 Typical을 선택하고 Next를 클릭한다.
    설치 준비가 완료되면 Install을 클릭하여 VMware Tools 설치를 시작하고 설치가 완료되면 Finish를 클릭한다. 아까와 마찬가지로 재시작 안내 창이 나타나면 바로 재시작하지 않고 No를 클릭하였다.


9. Windows Defender 방화벽에서 ICMP 허용

실습 환경에서는 다른 PC 또는 서버에서 Windows 10 VM으로 Ping 테스트를 해야 할 수 있다.
기본적으로 Windows Defender 방화벽에서 ICMP 요청이 차단될 수 있으므로, Ping 응답을 허용하기 위해 인바운드 규칙에서 ICMPv4 요청을 허용해야 한다.


  • 제어판 실행

    Windows + R을 눌러 실행 창을 열고 control을 입력한 뒤 확인을 클릭하여 제어판을 실행한다.


  • 보기 기준 변경

    제어판이 열리면 보기 기준을 작은 아이콘으로 변경한다.


  • Windows Defender 방화벽 실행

    목록에서 Windows Defender 방화벽을 클릭한다.


  • 고급 설정 열기

    왼쪽 메뉴에서 고급 설정을 클릭한다.


  • ICMPv4 인바운드 규칙 찾기

    고급 보안이 포함된 Windows Defender 방화벽 창에서 인바운드 규칙을 선택하고 목록에서 파일 및 프린터 공유(에코 요청 - ICMPv4-In) 규칙을 찾는다.


  • ICMPv4 규칙 사용 설정

    파일 및 프린터 공유(에코 요청 - ICMPv4-In) 규칙의 속성 창을 열고 사용함에 체크한 뒤 확인을 클릭한다.
    도메인과 개인 프로필 규칙 모두 해준다.


  • Ping 허용 상태 확인

    사용 상태가 예로 변경되면 외부에서 해당 Windows 10 VM으로 Ping 요청을 보낼 수 있다.


10. 전원 옵션 설정

가상 머신이 일정 시간 후 절전 모드로 전환되면 실습 중 연결이 끊기거나 서비스가 중단될 수 있다.
따라서 디스플레이 끄기와 절전 모드를 사용하지 않도록 설정한다.


  • 전원 옵션 실행

    제어판의 모든 제어판 항목에서 전원 옵션을 클릭한다.


  • 전원 관리 옵션 설정 변경

    사용 중인 전원 관리 옵션에서 설정 변경을 클릭한다.


  • 절전 설정 해제

    디스플레이 끄기와 컴퓨터를 절전 모드로 설정을 모두 해당 없음으로 변경한 후 변경 내용 저장을 클릭한다.


11. Windows Update 서비스 설정

실습 환경에서는 Windows Update가 자동으로 실행되면서 재부팅이나 네트워크 사용량 증가가 발생할 수 있다.
이를 방지하기 위해 Windows Update 서비스를 수동으로 제어할 수 있도록 설정하였다.


  • 서비스 관리 도구 실행

    Windows + R을 눌러 실행 창을 열고 services.msc을 입력한 뒤 확인을 클릭한다.


  • Windows Update 서비스 찾기

    서비스 목록에서 Windows Update를 찾는다.


  • 시작 유형 변경

    Windows Update 속성 창에서 시작 유형을 사용 안 함으로 변경한 후 로그온 탭으로 이동한다.


  • 로그온 계정 지정

    로그온 탭에서 계정 지정을 선택하고 찾아보기를 클릭한다.
    사용자 선택 창에서 고급 - 지금 찾기를 클릭한다.
    검색 결과에서 Guest 계정을 선택한 뒤 확인을 클릭한 후 암호와 암호 확인 값을 입력하고 다시 확인을 클릭한다.


12. 네트워크 IP 주소 설정

실습 환경에서 Windows 10 VM을 고정 IP로 사용하기 위해 네트워크 어댑터에 IP 주소를 수동으로 설정한다.


  • 네트워크 연결 창 실행

    Windows + R을 눌러 실행 창을 열고 ncpa.cpl을 입력한 뒤 확인을 클릭한다.


  • Ethernet0 IPv4에서 고정 IP 주소 입력

    네트워크 연결 창이 열리면 Ethernet0을 선택하고 속성을 클릭한 후 인터넷 프로토콜 버전 4(TCP/IPv4)를 선택, 속성을 클릭한다.
    IPv4 설정은 다음과 같이 입력하였다. 

      IP 주소: 10.0.0.101
  서브넷 마스크: 255.255.255.0
  기본 게이트웨이: 10.0.0.254
  기본 설정 DNS 서버: 168.126.63.1
  보조 DNS 서버: 8.8.8.8

    설정이 완료되면 확인을 클릭하여 네트워크 설정을 저장하고 시스템을 재부팅한다.
