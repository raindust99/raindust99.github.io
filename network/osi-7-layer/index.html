OSI 7계층 정의

OSI 7계층(Open Systems Interconnection)은 네트워크 통신을 7단계로 나눈 표준 모델이다.

계층 구조

(아래→위)
물리 → 데이터링크 → 네트워크 → 전송 → 세션 → 표현 → 응용

1. 물리 계층 (Physical Layer)

  • 실제 데이터 전송 (전기/광 신호)
  • 케이블, 허브 등 물리적 요소

단위: 비트(Bit)

2. 데이터링크 계층 (Data Link Layer)

  • MAC 주소 기반으로 인접 노드 간 전달
  • 오류 검출 및 수정

단위: 프레임(Frame)
장비: 스위치

3. 네트워크 계층 (Network Layer)

  • IP 기반으로 최종 목적지까지 전달
  • 경로 선택 (라우팅)
  • IP 주소 관리

단위: 패킷(Packet)
장비: 라우터
프로토콜: IP, ICMP

4. 전송 계층 (Transport Layer)

  • 데이터 신뢰성 보장
  • 오류 제어, 흐름 제어
  • 세그먼트 단위로 데이터 분할/재조립

단위: 세그먼트(Segment)
프로토콜: TCP, UDP

5. 세션 계층 (Session Layer)

  • 연결(세션) 생성, 유지, 종료
  • 통신 동기화
  • 동기화 체크포인트 설정 (중간부터 재전송 가능)

단위: 데이터(Data)
예시: 로그인 세션 유지

6. 표현 계층 (Presentation Layer)

  • 데이터 형식 변환
  • 암호화 / 복호화
  • 압축 / 해제

단위: 데이터(Data)
예시: HTTPS 암호화 처리

7. 응용 계층 (Application Layer)

  • 사용자와 직접 상호작용
  • 네트워크 서비스 제공

단위: 데이터(Data)
프로토콜: HTTP, HTTPS, FTP, SMTP, DNS
예시: 웹 브라우저에서 사이트 접속

핵심 요약

  • 하위 계층(1~3계층): 실제 데이터 전송 담당 (물리, MAC, IP 기반 통신)
  • 중간 계층(4계층): 종단 간 통신 제어 핵심 계층
    • 신뢰성 보장 (TCP)
    • 비연결 통신 (UDP)
    • 포트 번호 기반 프로세스 간 통신
    • 흐름 제어 / 오류 제어 수행
  • 상위 계층(5~7계층): 사용자 서비스 및 데이터 처리 담당
